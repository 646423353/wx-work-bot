设计系统概述

本设计系统为企业微信消息监控助手 MVP 版本提供完整的设计规范，确保产品视觉一致性和用户体验统一性。

专业可信赖

采用企业微信品牌蓝色系，传达专业、可信赖的品牌形象

高效监控

清晰的层级结构和信息架构，支持高效的消息监控工作流

警示优先

红/橙/绿颜色编码系统，确保未回复消息和告警信息醒目可见

模块化设计

可复用的组件库和一致的交互模式，降低开发成本

状态指示色使用规范

正常 (Normal)

使用 Success-500 (#52c41a)

应用场景：正常运营、已回复、响应达标

警告 (Warning)

应用场景：待处理、接近超时、需要注意

使用 Danger-500 (#ff4d4f)

应用场景：未回复超时、严重告警、异常状态

信息 (Info)

使用 Primary-500 (#0073e6)

应用场景：一般信息、进行中、待关注
